# Interaction Plan — E-Commerce Launchpad

This plan maps interaction types to lesson intent while avoiding over-quizzing.

## Recap Quiz Placement
- One recap quiz at the end of each module (12 total), each focused on transfer and decision quality.

## Instructional Logic by Lesson Pattern
- **Lesson 1 (concept foundations)**: interactive video + inline knowledge checks.
- **Lesson 2 (framework/decisions)**: reflection + branching scenario + short end check.
- **Lesson 3 (project/execution)**: worksheet/template/checklist + AI critique + artifact submission.
